## About the Project

The **E-Commerce App** is a full-stack web application designed to replicate a real-world online shopping experience.  
Users can browse products, manage carts, complete purchases, and authenticate securely. An integrated admin dashboard provides full control over products, categories, hero slides, settings, orders, and users.

This documentation highlights the architecture, features, and purpose of the project without exposing source code.

---

## Key Features

### User Features
- Product catalog with categories, filtering, and dynamic product pages
- Shopping cart with quantity management
- Checkout flow and order confirmation
- User authentication (registration, login, profile)
- Responsive UI for mobile and desktop

### Admin Dashboard
- Product and category management
- Hero slide and content management
- Order processing and status updates
- User and role management
- Platform settings and CMS-like controls

### Technical Features
- REST API architecture
- Authentication using access & refresh tokens
- HttpOnly cookie token handling
- Validation and structured error handling
- Scalable state management (Redux Toolkit)
